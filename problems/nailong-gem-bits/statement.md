# 奶龙的魔法水晶

## 题目描述

奶龙在魔法森林里发现了一块神奇的水晶，这块水晶包含 $30$ 层魔法能量层（编号从第 $0$ 层到第 $29$ 层）。小七告诉奶龙，这块水晶的总能量值可以用一个非负整数 $x$ 表示，而每一层的能量状态对应 $x$ 的二进制表示中的某一位：
- 如果第 $i$ 层有能量，则 $x$ 的二进制表示中第 $i$ 位为 $1$
- 如果第 $i$ 层没有能量，则 $x$ 的二进制表示中第 $i$ 位为 $0$

（这里的第 $i$ 位是指从右往左数，最右边是第 $0$ 位）

小七想要研究水晶的某些层次，她会进行 $n$ 次观测，每次观测她会：
1. 选择一个能量值 $x$（表示水晶当前状态）
2. 选择一个层次范围 $[l, r]$（表示观测第 $l$ 层到第 $r$ 层）
3. 询问奶龙：在这个范围内，有多少层是有能量的？

请帮助奶龙回答小七的每一次询问。

## 输入格式

第一行包含一个正整数 $n$，表示观测次数。

接下来 $n$ 行，每行包含三个非负整数 $x$, $l$, $r$，表示一次观测：水晶的能量值为 $x$，观测第 $l$ 层到第 $r$ 层（包含第 $l$ 层和第 $r$ 层）。

## 输出格式

输出 $n$ 行，每行包含一个整数，表示对应观测中有能量的层数。

## 样例输入 1
```
3
7 0 2
15 1 3
1000000000 10 20
```

## 样例输出 1
```
3
3
6
```

## 样例输入 2
```
4
5 0 0
5 1 1
5 2 2
5 0 2
```

## 样例输出 2
```
1
0
1
2
```

## 提示

**样例 1 解释：**
- 第一次观测：$7$ 的二进制表示是 $111$，第 $0$ 层到第 $2$ 层分别是 $1, 1, 1$，共 $3$ 层有能量。
- 第二次观测：$15$ 的二进制表示是 $1111$，第 $1$ 层到第 $3$ 层分别是 $1, 1, 1$，共 $3$ 层有能量。
- 第三次观测：$1000000000$ 的二进制表示是 $111011100110101100101000000000$，第 $10$ 层到第 $20$ 层中有 $6$ 层有能量。

**样例 2 解释：**
- $5$ 的二进制表示是 $101$：
  - 第 $0$ 层（最右边）：$1$，有能量
  - 第 $1$ 层：$0$，无能量
  - 第 $2$ 层：$1$，有能量
- 因此四次观测的答案分别是 $1, 0, 1, 2$。

**数据范围：**

对于 $100\%$ 的数据，保证 $1 \le n \le 1000$，$0 \le x \le 10^9$，$0 \le l \le r \le 29$。
