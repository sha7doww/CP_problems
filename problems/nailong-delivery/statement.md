# 变色“龙”邮差

## 题目背景

变色龙邮差生活在一个由 $n$ 个岛屿组成的群岛上。这些岛屿之间由 $m$ 条单向桥梁连接。每条桥梁都有其特定的颜色（红、绿、蓝）和通行时间。

邮差自身也有一个颜色状态。它的颜色会随着过桥而发生改变，而更麻烦的是：**桥梁的通行规则和通行时间，竟然取决于邮差当前的颜色！**

等等，这个变色龙邮差……怎么长得有点像奶龙？

没错，**奶龙**就是这位变色龙邮差！虽然它是一只可爱的小龙，但在这个神奇的群岛上，它获得了变色龙的能力——身体颜色会随着环境不断变化。

## 题目描述

奶龙初始在岛屿 $1$，身体颜色为**红色**（用 $0$ 表示）。它需要把快递送到岛屿 $n$。

每条桥梁由五个参数描述：起点 $u$、终点 $v$、桥的颜色 $c$（$0$ 红、$1$ 绿、$2$ 蓝）、基础时间 $t$、倍率因子 $k$。

当奶龙想要通过一条桥梁时，必须遵守以下规则：

1. **同色隐身**：变色龙的本能！如果奶龙当前的颜色与桥的颜色**相同**，它会本能地隐身，然后……一脚踩空掉进海里。所以，**同色时禁止通行**。

2. **颜色突变**：一旦踏上桥梁，奶龙的身体颜色会立即变成**第三种颜色**——既不是当前颜色，也不是桥的颜色。例如：奶龙是红色（$0$），桥是绿色（$1$），过桥后奶龙变为蓝色（$2$）。

3. **动态耗时**：通过桥梁的时间取决于**变色后**的新颜色 $c'$。实际耗时为 $t + k \times c'$。

请计算奶龙从岛屿 $1$ 到岛屿 $n$ 所需的**最短总时间**。

## 输入格式

第一行包含两个正整数 $n$ 和 $m$，分别表示岛屿数量和桥梁数量。

接下来 $m$ 行，每行包含五个整数 $u, v, c, t, k$，表示一条从岛屿 $u$ 到岛屿 $v$ 的单向桥梁，桥的颜色为 $c$（$0$ 红、$1$ 绿、$2$ 蓝），基础时间为 $t$，倍率因子为 $k$。

**数据范围：**

对于 $100\%$ 的数据，保证 $1 \le n \le 10^5$，$1 \le m \le 2 \times 10^5$，$1 \le u, v \le n$，$0 \le c \le 2$，$1 \le t \le 10^6$，$0 \le k \le 10^6$。

## 输出格式

输出一个整数，表示奶龙从岛屿 $1$ 到岛屿 $n$ 的最短时间。

如果无法到达，输出 `-1`。

## 样例输入 1

```
4 4
1 2 1 10 2
1 2 2 20 1
2 3 0 5 2
3 4 1 7 3
```

## 样例输出 1

```
37
```

## 样例输入 2

```
3 2
1 2 1 10 5
2 3 0 8 3
```

## 样例输出 2

```
31
```

## 提示

**样例 1 解释：**

奶龙初始在岛屿 1，身体是红色（$0$）。

贪心的奶龙想走看起来更快的第一条桥（绿色桥，基础时间 $10$）：
- 岛屿 1 → 2：红色奶龙走绿色桥，变为蓝色，耗时 $10 + 2 \times 2 = 14$
- 岛屿 2 → 3：蓝色奶龙走红色桥，变为绿色，耗时 $5 + 2 \times 1 = 7$
- 岛屿 3 → 4：绿色奶龙遇到绿色桥……**扑通！掉海里了！**

奶龙游回岸边，决定换一条路：
- 岛屿 1 → 2：红色奶龙走蓝色桥，变为绿色，耗时 $20 + 1 \times 1 = 21$
- 岛屿 2 → 3：绿色奶龙走红色桥，变为蓝色，耗时 $5 + 2 \times 2 = 9$
- 岛屿 3 → 4：蓝色奶龙走绿色桥，变为红色，耗时 $7 + 3 \times 0 = 7$

总时间为 $21 + 9 + 7 = 37$。快递送达！

**样例 2 解释：**

- 岛屿 1 → 2：红色奶龙走绿色桥，变为蓝色，耗时 $10 + 5 \times 2 = 20$
- 岛屿 2 → 3：蓝色奶龙走红色桥，变为绿色，耗时 $8 + 3 \times 1 = 11$

总时间为 $20 + 11 = 31$。
