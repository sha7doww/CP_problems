# 奶龙的量子纠缠构造 (Nailong's Quantum Entanglement Construction)

## 题目描述

![](./assets/figure.png)

奶龙最近迷上了量子计算，在小七（$\texttt{sha7dow}$）的 LLL 实验室里研究量子纠缠现象。

在量子世界中，有一种神奇的"纠缠序列"：给定一个正整数数组 $A$，如果某个连续子数组 $[l, r]$ 的所有元素异或（XOR）起来等于 0，就称这个子数组产生了一次"量子纠缠"。

小七给奶龙出了一道难题：请你构造一个长度为 $N$ 的正整数数组 $A$，使得恰好有 $K$ 个连续子数组的异或和为 0。

形式化地说，需要满足：
- 对于所有 $1 \le i \le N$，有 $A_i \ge 1$（数组中不包含 0）
- 满足 $A_l \oplus A_{l+1} \oplus \cdots \oplus A_r = 0$ 的数对 $(l, r)$（其中 $1 \le l \le r \le N$）的数量恰好为 $K$

如果存在多个满足条件的数组，输出任意一个。如果不存在这样的数组，输出 `-1`。

奶龙挠了挠头："这也太难了吧！"你能帮帮它吗？

## 输入格式

一行包含两个整数 $N$ 和 $K$，分别表示数组长度和目标纠缠数。

**数据范围：**
对于 $100\%$ 的数据，保证 $1 \le N \le 1000$，$0 \le K \le 500000$，$1 \le A_i \le 10^9$。

**时间限制：** 2.0 秒 | **内存限制：** 256 MB

## 输出格式

- 如果有解，输出一行 $N$ 个正整数，表示数组 $A$，每个元素之间用空格分隔。
- 如果无解，输出 `-1`。

## 样例输入 1

```
4 1
```

## 样例输出 1

```
1 2 3 4
```

## 样例输入 2

```
3 3
```

## 样例输出 2

```
-1
```

## 提示

**样例 1 解释：**
数组 $A = [1, 2, 3, 4]$ 中，只有子数组 $[1, 3]$ 的异或和为 0（$1 \oplus 2 \oplus 3 = 0$），所以纠缠数为 1。

**样例 2 解释：**
可以证明，对于 $N = 3$，不存在任何正整数数组使得恰好有 3 个子数组的异或和为 0。
