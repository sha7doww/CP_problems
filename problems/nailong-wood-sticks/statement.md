# 奶龙的木棒

## 题目描述

奶龙最近收集了很多木棒，小七想用这些木棒来搭建一个漂亮的小屋。

奶龙一共有 $n$ 根木棒，第 $i$ 根木棒的长度是 $L_i$ 厘米。小七需要恰好 $k$ 根长度完全相同的木棒来搭建小屋的框架。

为了帮助小七，奶龙可以将木棒切成更短的木棒（切割次数不限），但是不能将多根木棒拼接起来。奶龙想知道，这 $k$ 根相同长度的木棒，最长可以是多少厘米？

注意：切割后的木棒长度必须是整数厘米。

## 输入格式

第一行包含两个正整数 $n$ 和 $k$，分别表示木棒的数量和小七需要的木棒根数。

第二行包含 $n$ 个正整数 $L_1, L_2, \ldots, L_n$，表示每根木棒的长度（单位：厘米）。

## 输出格式

输出一个整数，表示 $k$ 根相同长度木棒的最大长度（单位：厘米）。

如果无法获得 $k$ 根木棒，输出 $0$。

## 样例输入 1

```
4 11
802 743 457 539
```

## 样例输出 1

```
200
```

## 样例输入 2

```
3 5
100 50 30
```

## 样例输出 2

```
30
```

## 提示

**样例 1 解释：**

奶龙可以这样切割：
- 第 1 根木棒（802 厘米）切成 4 根 200 厘米的木棒，剩余 2 厘米
- 第 2 根木棒（743 厘米）切成 3 根 200 厘米的木棒，剩余 143 厘米
- 第 3 根木棒（457 厘米）切成 2 根 200 厘米的木棒，剩余 57 厘米
- 第 4 根木棒（539 厘米）切成 2 根 200 厘米的木棒，剩余 139 厘米

总共得到 $4 + 3 + 2 + 2 = 11$ 根长度为 200 厘米的木棒。

**样例 2 解释：**

奶龙可以这样切割：
- 第 1 根木棒（100 厘米）切成 3 根 30 厘米的木棒，剩余 10 厘米
- 第 2 根木棒（50 厘米）切成 1 根 30 厘米的木棒，剩余 20 厘米
- 第 3 根木棒（30 厘米）恰好是 1 根 30 厘米的木棒

总共得到 $3 + 1 + 1 = 5$ 根长度为 30 厘米的木棒。

**数据范围：**

对于 $100\%$ 的数据，保证 $1 \le n \le 10^4$，$1 \le k \le 10^4$，$1 \le L_i \le 10^8$。
