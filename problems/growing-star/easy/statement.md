# 成长之星（入门）

## 题目描述

在一个充满活力的城市里，有一群年轻人，他们都有着各自的理想和目标。市长决定举办一场竞赛，寻找那些在学习和工作中表现出持续进步的年轻人。竞赛的规则很简单：谁能够展现出最长的"递增"轨迹，谁就能获得"城市之星"的称号。

小徐就是年轻人之一，他带着自己最近一段时间的评分，来参与这场竞赛。聪明的你能帮他找找最长的**连续上升子序列**的长度吗？

**连续上升子序列**是指在原序列中连续的一段，且每个元素都严格大于前一个元素。

## 输入格式

输入的第一行有 1 个整数 $n$，表示一共有 $n$ 个整数评分。

第二行有 $n$ 个整数，表示小徐的 $n$ 个评分。

**数据范围：**

对于 $100\%$ 的数据，保证 $1 \le n \le 1000$，$1 \le a_i \le 10^6$。

## 输出格式

输出一行，这一行只包含一个整数，表示最长的连续上升子序列的长度。

## 样例输入 1

```
6
1 3 5 4 7 8
```

## 样例输出 1

```
3
```

## 样例输入 2

```
5
5 4 3 2 1
```

## 样例输出 2

```
1
```

## 提示

**样例 1 解释：**

最长的连续上升子序列有两个：$[1, 3, 5]$ 和 $[4, 7, 8]$，长度都是 3。

**样例 2 解释：**

序列是严格递减的，所以最长的连续上升子序列长度为 1（任意单个元素）。
