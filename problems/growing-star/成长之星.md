# 成长之星

在一个充满活力的城市里，有一群年轻人，他们都有着各自的理想和目标。市长决定举办一场竞赛，寻找那些在学习和工作中表现出持续进步的年轻人。竞赛的规则很简单：谁能够展现出最长的“递增”轨迹，谁就能获得“城市之星”的称号。

小徐就是年轻人之一，他带着自己最近一段时间评分，来参与这场竞赛。聪明的你能帮他找找最长的连续上升子序列的长度吗？

**输入格式**

输入的第一行有1个整数$n$，表示一共有$n$个整数评分

第二有$m$个整数，表示小徐的n个评分。

**输出格式**

输出一行，这一行只包含一个整数，表示最长的连续上升子序列的长度。



## 一、入门等级

- 解题思想：暴力解法
- 复杂度： 时间复杂度O(n*n) ; 空间复杂度O(n)
- C语言实现

```c++
#include <stdio.h>

int longest(int scores[], int n) {
    if (n == 0) return 0;

    int max_length = 1;

    for (int i = 0; i < n; i++) {
        int current_length = 1;
        for (int j = i + 1; j < n && scores[j] > scores[j - 1]; j++) {
            current_length++;
        }
        if (current_length > max_length) {
            max_length = current_length;
        }
    }

    return max_length;
}

int main() {
    int n;
    scanf("%d", &n);

    int scores[n];
    for (int i = 0; i < n; i++) {
        scanf("%d", &scores[i]);
    }

    int result = longest(scores, n);
    printf("%d\n", result);

    return 0;
}
```

## 二、一般等级

- 算法思路：动态规划。(1)状态表示：f[i]表示从第一个数开始算，以ratings[i]为结尾的最大上升子序列。(2)状态计算：f[i]=max(f[i], f[i-1])。最后在f[i]中找最大值。
- 复杂度：时间复杂度O(n^2)，空间复杂度O(n)。
- C++实现：

```c++
#include <iostream>
#include <cstdio>
#include <cstring>
#include <algorithm>

using namespace std;

const int N  = 1010;

int n, f[N], ratings[N];

int main()
{
    scanf("%d", &n);
    
    for (int i = 1; i <= n; ++i) scanf("%d", &ratings[i]);
    
    
    for (int i = 1; i <= n; ++i)
    {
        f[i] = 1;
        for (int j = 1; j < i; ++j)
            if (ratings[i] > ratings[j])
                f[i] = max(f[i], f[j] + 1);
    }
                
    
    int res = 0;
    for (int i = 1; i <= n; ++i)
        res = max(res, f[i]);
        
    printf("%d", res);
    
    return 0;
}

```

## 三、困难等级

- 算法思路：优化后的动态规划，对于每个值，使用二分查找在 数组中找出第一个大于等于该值的位置 。
- 复杂度：时间复杂度O(nlogn)，空间复杂度O(n)。
- C++实现：

```
#include <stdio.h>
int binarySearch(int ends[], int left, int right, int key) {
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (ends[mid] < key) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    return left;
}
int longest(int scores[], int n) {
    if (n == 0) return 0;

    int ends[n];
    int length = 0; 

    for (int i = 0; i < n; i++) {
        int pos = binarySearch(ends, 0, length, scores[i]);
        ends[pos] = scores[i];
        if (pos == length) {
            length++;
        }
    }

    return length;
}

int main() {
    int n;

    scanf("%d", &n);
    int scores[n];
    for (int i = 0; i < n; i++) {
        scanf("%d", &scores[i]);
    }

    int result = longest(scores, n);
    printf("%d\n", result);
    return 0;
}
```
