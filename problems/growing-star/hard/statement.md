# 成长之星（困难）

## 题目描述

在一个充满活力的城市里，有一群年轻人，他们都有着各自的理想和目标。市长决定举办一场竞赛，寻找那些在学习和工作中表现出持续进步的年轻人。竞赛的规则很简单：谁能够展现出最长的"递增"轨迹，谁就能获得"城市之星"的称号。

小徐就是年轻人之一，他带着自己最近一段时间的评分，来参与这场竞赛。聪明的你能帮他找找**最长上升子序列**的长度吗？

**最长上升子序列**是指从原序列中选出若干个元素（不要求连续），保持它们在原序列中的相对顺序，且每个元素都严格大于前一个元素，使得选出的元素个数最多。

## 输入格式

输入的第一行有 1 个整数 $n$，表示一共有 $n$ 个整数评分。

第二行有 $n$ 个整数，表示小徐的 $n$ 个评分。

**数据范围：**

对于 $100\%$ 的数据，保证 $1 \le n \le 10^5$，$1 \le a_i \le 10^9$。

## 输出格式

输出一行，这一行只包含一个整数，表示最长上升子序列的长度。

## 样例输入 1

```
6
1 3 5 4 7 8
```

## 样例输出 1

```
5
```

## 样例输入 2

```
5
5 4 3 2 1
```

## 样例输出 2

```
1
```

## 提示

**样例 1 解释：**

最长上升子序列为 $[1, 3, 5, 7, 8]$ 或 $[1, 3, 4, 7, 8]$，长度为 5。

**样例 2 解释：**

序列是严格递减的，所以最长上升子序列长度为 1（任意单个元素）。
