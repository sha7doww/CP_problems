# LLL 实验室招新解密挑战

## 题目背景

LLL 实验室（DL，RL，NL）是学校最顶尖的 AI 研究机构，下设三个研究方向：

- **DL 方向**（Deep Learning）：深度学习
- **RL 方向**（Reinforcement Learning）：强化学习
- **NL 方向**（NaiLong Learning）：奶龙学习

今年 LLL 实验室正式启动招新计划。dl（Memory）、fgw（gui）、zfs（T-mac）听闻这个消息后，立即前往报名参加考核。自然地，dl 率先抢走了 RL 方向，zfs 紧随其后地选择了 NL 方向，gui 选择了时薪一龟元的 DL 方向。

考核官给 zfs 出了一道经典的密码学数学问题："在 RSA 加密算法中，如果你知道了某些关键参数，能否快速找出两个解？"

这正是 zfs 证明自己在 NL 方向上实力的绝佳机会！

## 题目描述

给定三个正整数 $n, e, d$，请求出两个正整数 $p, q$，使得：
- $n = p \times q$
- $e \times d = (p - 1)(q - 1) + 1$

## 输入格式

一行三个正整数 $n, e, d$，用空格分隔。

## 输出格式

如果存在满足条件的 $p, q$，输出一行两个正整数 $p, q$，用空格分隔。

为使输出统一，你应当保证 $p \leq q$。

如果无解，请输出 `NO`。

## 样例输入 1

```
770 5 77
```

## 样例输出 1

```
2 385
```

## 样例输入 2

```
633 211 1
```

## 样例输出 2

```
NO
```

## 提示

**样例 1 解释**：$n = 2 \times 385 = 770$，$e \times d = 5 \times 77 = 385 = (2-1)(385-1) + 1$，满足所有条件。

**样例 2 解释**：不存在满足条件的正整数 $p, q$。

**解题思路提示**：

设 $m = p + q$，则 $m = n - e \times d + 2$。

已知两数之和 $m$ 与两数之积 $n$，可以构造一元二次方程 $x^2 - mx + n = 0$。

根据求根公式：$x = \frac{m \pm \sqrt{m^2 - 4n}}{2}$

判断是否有解需要检查：
- 判别式 $\Delta = m^2 - 4n$ 是否为非负完全平方数
- 两个根是否都是正整数

**数据范围**：

对于 $100\%$ 的数据，$1 \leq n \leq 10^{18}$，$1 \leq e \times d \leq 10^{18}$，$1 \leq m \leq 10^9$（其中 $m = n - e \times d + 2$）。
